SOFTWARE DEVELOPMENT LIFE CYCLE (SDLC) ANALYSIS

The Student weekly planning system is a web-based application designed to help students plan out their activities for the week, add time for the activities and recieve reminders to complete those task,and congratulates users for completing task. It helps student with poor management skills to be more organized and carry out tasks efficiently.

  Requirement Analysis 
  
Functional Requirements
* Add weekly tasks
* Assign start and end time to tasks
* Display reminders at scheduled time
* Mark tasks as completed
* Display congratulatory message
* 
Non-Functional Requirements
* Easy to use
* Fast response
* Reliable reminders

   System Design
  
Task Data Structure
* taskId
* taskTitle
* taskDay
* startTime
* endTime
* taskStatus 

System Components
* Task Manager
* Reminder Module
* Completion Checker
* Notification Module

     Implementation
  
The application was implemented using HTML, CSS, and JavaScript
